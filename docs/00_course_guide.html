Course Guide

Welcome to LLM Lessons — a hands-on mini classroom where you'll build and understand a tiny version of the same technology behind ChatGPT.

You won't just read about AI. You'll train a real model, watch it learn, and talk to it.

What This Project Does

There is a fictional pizzeria called Slice Pizza. We trained a small AI on pizza conversations. When you ask it "What pizza do you recommend?", the model generates the answer character by character — just like a real language model does. No cheating, no hardcoded answers.

How to Study

Follow this order. Each step builds on the last:

  1. Read this guide (you're here).
  2. Introduction — what we're building and why.
  3. How LLMs Work — the big picture of how AI language models work.
  4. Simple Context Model — how a conversation becomes model input.
  5. Tokenization through Limitations of the Model — one concept at a time.
  6. File-by-File Lessons — connect every concept back to real code.
  7. Open the files in app/ and explore.
  8. Run the tests, train the model, ask it questions.

Commands

Install dependencies:

pip install -r requirements.txt

Train the model:

python app/train.py

Ask the trained model a pizza question:

python app/infer.py --prompt "What pizza do you recommend?"

Ask it something it wasn't trained on:

python app/infer.py --prompt "What is the capital of France?"

See the full raw generated conversation:

python app/infer.py --raw-model --prompt "What pizza do you recommend?"

Start the API server:

uvicorn app.api:app --reload

Run all tests:

python -m pytest

The Core Idea

The model learns one simple skill, over and over:

Given the previous characters, predict the next character.

Training example:

input:  <|user|> What piz
target: |user|> What pizz

The target is just the input shifted one character to the right. This simple trick is the same idea behind every GPT-style model — just at a tiny scale.
